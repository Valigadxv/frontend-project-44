## Descriptopn

"Brain Games" - are five logical mini-games to test and develop your thinking.
To win, you need to give three correct answers in a row, and to lose, one wrong is enough.

### Setting up

```
npm i
```
```
npm link
```


### How to run the games 
```
brain-even  
```
```
brain-calc  
```
```
brain-prime  
```
```
brain-gcd  
```
```
brain-progression  
```
